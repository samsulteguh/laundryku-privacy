Privacy Policy LaundryKu

Terakhir diperbarui: Maret 2026

LaundryKu ("kami") berkomitmen untuk melindungi privasi pengguna aplikasi LaundryKu.

Informasi yang Kami Kumpulkan

  • Nama dan alamat email untuk autentikasi
  • Data laundry (nama usaha, alamat, nomor WhatsApp)
  • Data order dan pelanggan yang Anda masukkan
  • Informasi perangkat untuk keamanan akun

Cara Kami Menggunakan Informasi

  • Menyediakan layanan manajemen laundry
  • Autentikasi dan keamanan akun

  • Meningkatkan kualitas aplikasi

Penyimpanan Data

Data disimpan secara aman menggunakan Firebase (Google Cloud) dengan enkripsi standar industri.

Berbagi Data

Kami tidak menjual atau berbagi data pribadi Anda kepada pihak ketiga, kecuali untuk keperluan pembayaran melalui Midtrans.

Keamanan

Kami menggunakan langkah-langkah keamanan industri untuk melindungi data Anda termasuk enkripsi SSL dan Firebase Security Rules.
